# CNN Architecture To Train MNIST Dataset Using PyTorch

A convolutional neural network (CNN) implemented in PyTorch to classify handwritten
digits from the MNIST dataset.

## Overview

This project builds, trains, and evaluates a CNN that learns to recognize the
digits 0–9 from 28×28 grayscale images in the MNIST dataset.

## Requirements

- Python 3.8+
- PyTorch
- torchvision

```bash
pip install torch torchvision
```

## Usage

```bash
python train.py
```
